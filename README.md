# ProjectBridge

A student-built project collaboration platform for Foothill College — designed for first-gen, international, and underrepresented students.

Built with React + Vite. Hosted on Cloudflare Pages.

## Local Development

```bash
npm install
npm run dev
```

## Deploy to Cloudflare Pages

This project is configured for Cloudflare Pages with automatic deploys on push.

- **Build command:** `npm run build`
- **Build output directory:** `dist`
- **Node.js version:** 18+
